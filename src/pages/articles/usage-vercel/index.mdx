Vercel 是我用过的最好用的网站托管服务。它和 github page 类似，但功能更加强大：

1. github 仓库对 vercel 进行授权后，能够获得最优雅的发布体验
2. vercel 支持部署 静态网站，serverless 接口。这就意味着前后端可以完全用 vercel 接管
3. 部署成功后 vercel 免费提供一个二级域名；还可以设置自定义域名
4. vercel 会自动匹配一个 SSL 证书
5. 支持自动配置的模板达 31 种之多，也就是部署时可以达到 0 配置
6. 提供 production 和 preview 两种环境的版本隔离

对个人开发者可以说是非常友好且实用
